Parameters:

Returns:

    num x any value but 0 will stop the Sniffer
TbxSniffer:start ([cb])
Start sniffing and calls the callback function for each new packet. Will never return unless stop is called from another thread or if it was started in non-blocking mode.

Parameters:

  • cb function a function callback, see sniffer_callback, to operate in blokcing mode. Omit to be non-blocking. (optional)
